  2. 在下面输入你的GitHub Token
  3. 测试连接确保配置正确

第一步：创建 GitHub Personal Access Token

1. 访问 GitHub Token 设置页面

2. 点击 "Generate new token" > "Generate new token (beta)"

3. 配置Token信息：

Token name: BloodMoney Comments System Description: 用于BloodMoney游戏网站评论系统 Expiration: 90 days Resource owner: jiang Repository access: Selected repositories -> bloodmoneygame Repository permissions: ✅ Contents: Write ✅ Metadata: Read ✅ Actions: Write (可选)

4. 点击 "Generate token" 并复制生成的token
