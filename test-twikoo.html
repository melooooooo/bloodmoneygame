This page is for testing the Twikoo comment system integration before deploying to production.

Setup Steps:

  1. Deploy the Twikoo Worker to Cloudflare:
    • Run npx wrangler d1 create twikoo-bloodmoney
    • Update database ID in twikoo-deploy/wrangler.toml
    • Run npx wrangler d1 execute twikoo-bloodmoney --remote --file=./schema.sql
    • Run npx wrangler r2 bucket create twikoo-bloodmoney
    • Deploy with npx wrangler deploy --minify
  2. Update the Worker URL in rs/js/twikoo-init.js
  3. Refresh this page to see the comment system
